使用eclipse新建osgi工程
2019-11-19

前提 java环境

1.eclipse安装(注意：推荐上面连接 必须是企业版 不然需要自己配插件之类的)

软件直达网址

2.新建工程

file->new->project->plugin

3.运行配置

打开运行三角旁边的小三角 选择run config

依次选择workplace中你新建的工程和下面四个依赖

点击apply和run

4.恭喜你，到现在已经完成了新建 ，在控制台中你应该可以看到“hello world”了。

输入ss可以查看现在组件的运行状态

5.如果你不幸的报错了 请仔细检查你的步骤和报错信息，应该是Bundle依赖错误引起的，参考步骤3或者移步www.baidu.com
